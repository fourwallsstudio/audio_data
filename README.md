# Audio Data

## dependencies
pydub  
librosa  

## use
create directories:

/split

/house/full  
/house/split  

/techno/full  
/techno/split  



put audio files in main directory and run `python split_audio.py`  
use only mp3 or wav files  
put samples in /genre/split   
put full files in /genre/full
